# Master Feature Inventory

Single source of truth for Phase 1 tracking. Each row is a dashboard summary — full evidence and documentation for each feature lives in its per-area file (linked in the "Notes" column once that file exists). Update this table every time a HAR is analyzed or a Figma export is reviewed; don't let detail accumulate here that belongs in a per-area file instead.

**Status legend:** `Confirmed` = verified from a capture/export. `Partial` = some verified, some not. `Not yet captured` = no evidence gathered yet (not "unknown behavior," just "haven't looked"). `N/A` = doesn't apply to that column for this feature.

**Current priority order (superseding the original `CLAUDE.md`-seeded numbers where they conflict, per explicit direction 2026-07-07): Activity Feed → Members Directory → Profile/XProfile → Friends → Groups → Notifications → Courses.** Messaging and Authentication are done/ongoing infrastructure, not part of this ordered list. K54 AI Assistant is confirmed a first-class feature with its own dedicated audit **scheduled after the 7 core features complete** — not folded into the core order. Marketplace remains pending product clarification. Live Streaming/VOD and Events (newly discovered, not part of the original scope) have been moved to `future-discoveries.md` so they don't interrupt the core audit — see that file if they become relevant later.
